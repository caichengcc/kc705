Detailed Description

This structure contains the logiCVC information gained from hardware configuration. All the elements are OsU32 to simplify usage.

Field Documentation

OsU32 logiCvcConfig::ipMajorRevision

logiCVC IP core major revision, range (0:31)

OsU32 logiCvcConfig::ipMinorRevision

logiCVC IP core minor revision, range (0:31)

OsU32 logiCvcConfig::ipPatchRevision

logiCVC IP core patch revision, range (0:25), VALUES = (0=a,1=b,2=c,3=d,4=e,5=f,6=g,7=h,8=i,9=j,10=k,11=l,12=m,13=n,14=o,15=p,16=q,17=r,18=s,19=t,20=u,21=v,22=w,23=x,24=y,25=z)

OsU32 logiCvcConfig::Layer_AlphaMode[LOGICVC_MAX_LAYERS_PER_CVC]

Alpha blending mode on a layer, 0-layer, 1- pixel, 2- clut

OsU32 logiCvcConfig::Layer_Buffer_Offset[LOGICVC_MAX_LAYERS_PER_CVC]

Buffer offset expressed in horizontal lines starting from Buffer A offset, calculation: BufferB = BufferA + Buffer_offset, BufferC = BufferA + 2*Buffer_offset

OsU32 logiCvcConfig::Layer_BufferA_Offset[LOGICVC_MAX_LAYERS_PER_CVC]

Buffer A offset in expressed in horizontal lines starting from MemoryLowAddr

OsU32 logiCvcConfig::Layer_DataWidth[LOGICVC_MAX_LAYERS_PER_CVC]

Number of bits per pixel on a layer

OsU32 logiCvcConfig::Layer_Type[LOGICVC_MAX_LAYERS_PER_CVC]

Layer type, i.e. color format: RGB, YCbCr

OsU32 logiCvcConfig::MemoryHighAddr

Highest memory address visible from logiCVC device

OsU32 logiCvcConfig::MemoryLowAddr

Lowest memory address visible from logiCVC device

OsU32 logiCvcConfig::NumLayers

Number of layes supported on this CVC instance, currently CVC can support up to max 5 layers

OsU32 logiCvcConfig::ReadableRegs

If !=0 device registers are readable, if 0 only interrupt status register and double/triple video/CLUT buffer registers are readable

OsU32 logiCvcConfig::RegBaseAddr

logiCVC Register base address

OsU32 logiCvcConfig::RegHighAddr

logiCVC Last register address

OsU32 logiCvcConfig::RowStride

Row stride in number of pixels

OsU32 logiCvcConfig::UseBackground

Is last layer configured as background layer: 0- no, 1 -yes
