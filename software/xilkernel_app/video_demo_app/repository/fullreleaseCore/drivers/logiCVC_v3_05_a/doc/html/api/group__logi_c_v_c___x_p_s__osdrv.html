enum logiCVC_init_flagsE

This enum contains the flags that enable parts of logiCVC initialization code

Enumerator:
skipNone 

Noting is skipped

skipCvcInitRegisters 

Only init driver but doesn't write in any of logiCVC registers

skipCvcPowerControl 

Skip power sequence

skipClearVideoMemory 

Skip clearing video memory and CLUTs

skipEnableLayers 

Skip enabling layers

skipDisplayControl 

Skip calling display control functions logiCVC_plfInitCallbackT

skipAll 

Skip all but the logiCVC driver init

clearAllLayerBuffers 

Clear all buffers on all logiCVC layers, valid if skipClearVideoMemory is not set

logiCvcPointerT logiCVC_init ( OsSizeT  instanceId,
OsSizeT  displayType,
logiCVC_init_flagsE  flags,
logiCVC_plfInitCallbackT *  plfCallback 
)

Initializes a logiCVC instance. Uses a display type ID.

Parameters:
instanceIdis the unique id of the logCVC device. Values from 0 to (XPAR_LOGICVC_NUM_INSTANCES-1)
displayTypeis number defining logiCVC display type (e.g. LOGICVC_DT_VGA_640_480)
flagsare flags of logiCVC_init_flagsE type, and define different options
plfCallbackis pointer to structure containing callbacks for clock, display and backlight initialization. Pointer in this structure must be initialized to valid platform functions or set to NULL in which case they wont be called. plfCallback can also be set to NULL in which case 0 callback will be called.
Returns:
logiCVC instance pointer if everything starts up as expected
NULL if the init was unsuccessful
Note:
  • plfCallback structure must be initialized with platform function for clock, display and backlight initialization when logiCVC is not controlling the display
  • logiCVC background and layer transparent colors will be set to default values defined in this file
