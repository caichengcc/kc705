Driver initialization

Driver initialization steps:

  • initialize logiCVC instances with function logiCVC_Initialize()
  • initialize and start logiCVC with function logiCVC_Start(); sets display resolution
  • (optional) setting the background and transparent color with functions logiCVC_SetBackgroundColor() and logiCVC_SetTransparentColor()
  • enable layers with function logiCVC_LayerEnDis() (Enable = 1)


All of the forementioned, plus the display backlight and power handling, is unified in one function: logiCVC_init(). Alternatively, logiCVC_initDd() can be used. If any of these two functions is used, make sure to include logiCVC_init.h.
An example of driver initialization is given in logiCVC_example.c.
