Driver notes


Structure logiCVC_Config contains the logiCVC information gained from hardware configuration while the structure logiCVC_Layer contains the logiCVC layer information data.
When setting the transparent color of a layer the user has to make sure that he or she supplies 8, 16 or 24bpp color to the function depending on layer color depth. So, for 16bpp layer only lowest 2 bytes are used.
The background color is the transparent color in 24bpp RGB format. Background color will be displayed on the last layer if this is enabled in HW configuration.
Buffering can be double of triple. Number of buffer is define by the constant LOGICVC_NUM_BUFFERS. Switching between buffer is supported.
There are three elementary drawing functions available: logiCVC_DrawPixel(), logiCVC_FillRectangle() and logiCVC_CopyRectangle().

logiCVC LIMITATIONS:
The number of layers is limited to (LOGICVC_MAX_LAYERS_PER_CVC).
The logiCVC instance is the unique id of the logCVC device with values from 0 to (XPAR_LOGICVC_NUM_INSTANCES-1).
When setting the logiCVC layer output position (OutputPosition x_pos + width) must be less then logiCVC horizontal resolution and (OutputPosition y_pos + height) must be less then logiCVC Vertical resolution.
